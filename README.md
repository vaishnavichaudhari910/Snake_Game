# 🐍 Snake Game

A classic Snake Game built using JavaScript, where players control the snake to eat food, grow longer, and avoid collisions.
---

## 🎮 Features

* 🕹️ Smooth keyboard controls
* 🍎 Food generation system
* 📈 Score tracking
* 💀 Game over on collision (wall or self)
* ⚡ Fast and responsive gameplay

---

## 🛠️ Tech Stack

* HTML
* CSS
* JavaScript
